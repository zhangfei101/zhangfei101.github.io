# 动物栖息地与分布网站 - 图片替换指南

## 网站概述

这是一个展示动物栖息地与分布信息的静态网站，支持中英双语切换。

## 替换步骤

1. 准备好上述14张真实主题相关图片
2. 将图片文件命名为对应的文件名（如 habitat-map.jpg）
3. 将图片放入网站根目录下的 `images` 文件夹
4. 确保图片格式为 JPG 或 PNG
5. 重启网站服务器以查看更新
