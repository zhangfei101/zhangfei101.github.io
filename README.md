# 动物栖息地与分布网站

这是一个展示动物栖息地与分布信息的专业静态网站，支持中英双语切换，内容来源于 `Habitat & Range.md` 文件。

## 功能特点

- **中英双语切换** - 点击页面右上角的 EN/中 按钮可切换语言
- **响应式设计** - 适配桌面端、平板和移动端设备
- **专业内容** - 基于真实科学数据和研究
- **快速加载** - 静态资源优化，加载速度快
- **无障碍友好** - 符合无障碍标准的设计

## 内容结构

网站包含以下主要部分：

1. **全球猫科动物种群密度分布图**
   - 展示22种猫科动物的分布情况
   - 来自相机陷阱研究的679条种群密度记录

2. **保护状况**
   - 国际自然保护联盟(IUCN)红色名录相关信息
   - 物种灭绝风险评估数据

3. **人类活动对野生动物的影响**
   - 森林砍伐
   - 栖息地破碎化
   - 偷猎
   - 污染
   - 其他生态问题

4. **动物权利发展历程时间线**
   - 记录从公元前530年至今的重要事件
   - 人类对动物权益认知的演变

5. **保护行动**
   - 当前采取的保护措施
   - 未来的保护方向

## 技术架构

- **前端技术**: HTML5, CSS3, JavaScript ES6
- **设计风格**: 现代扁平化设计，绿色自然主题
- **兼容性**: 支持现代浏览器（Chrome, Firefox, Safari, Edge）
- **性能**: 优化的代码结构，快速加载

## 部署说明

### 本地预览

1. 确保已安装 Python
2. 在项目根目录运行：
   ```bash
   python -m http.server 8000
   ```
3. 访问 http://localhost:8000 查看网站

### 生产部署

可部署到任意静态网站托管服务：
- GitHub Pages
- Netlify
- Vercel
- AWS S3
- 传统Web服务器

## 国际化支持

网站完全支持中英文双语，语言切换功能实现：
- 即时切换，无需页面刷新

- 完整内容翻译
